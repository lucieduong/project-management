# Start here: Website design workflow

Use this folder when you’re ready to create a **website mockup** for a project in the Website Dev Pipeline. Everything is in order: do **Step 1**, then **Step 2**, then **Step 3**.

---

## What you’ll do

1. **Step 1** — Fill in your project page in Notion (client, keywords, competitors, design inputs).
2. **Step 2** — In Cursor, paste a short prompt and your project page link; copy the Figma Make prompt the AI gives you.
3. **Step 3** — Paste that prompt into Figma Make, generate the design, then save the Figma link back in Notion.
