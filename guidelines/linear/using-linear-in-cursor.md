# Using Linear in Cursor

Cursor is connected to your Linear workspace. You can **create**, **check**, and **edit** issues by talking to Cursor in plain language. You don’t need to use the Linear website or learn any technical steps.

---

## What you can do

### Create an issue

- Say: **"Create a Linear issue"** or **"I need to add an issue to Linear."**
- Cursor will ask you a few questions (one at a time), then create the issue in Linear for you.
- You’ll get a confirmation with the new issue (e.g. link or ID). No copy-paste needed.

**Tip:** You can also paste the contents of `issues/create-issue-prompt.md` into the chat to start the same flow.

---

### Check issues

- **"Show my Linear issues"** — Lists issues assigned to you.
- **"What issues are in [team name]?"** — Lists issues for that team.
- **"Show me issue [ID or title]"** — Opens one issue (e.g. "PROJ-42" or part of the title).
- **"Search Linear for [topic]"** — Finds issues that match.

---

### Edit an issue

- **"Update issue [ID or title]..."** — Tell Cursor what to change (e.g. "Update issue PROJ-42: set priority to High and add a note that the client confirmed the bug.").
- **"Change the status of PROJ-42 to In progress"** — Updates the issue state.
- You can ask to change: title, description, priority, status, assignee, labels, due date, or add a comment.

---

## What to have ready

- **Creating:** Know the **team** the issue belongs to (e.g. "Web", "Mobile"). Cursor can list your teams if you ask.
- **Editing:** The **issue ID** (e.g. PROJ-42) or enough of the title for Cursor to find it.

---

## If something goes wrong

- **"List my Linear teams"** — See team names so you can pick the right one when creating issues.
- **"What projects are in Linear?"** — See projects if you want to attach an issue to one.
- If Cursor doesn’t find an issue, try giving the exact ID (e.g. PROJ-42).

---

**Summary:** Say what you want in normal words ("create an issue", "show my issues", "update PROJ-42"). Cursor will use Linear for you and confirm what it did.
