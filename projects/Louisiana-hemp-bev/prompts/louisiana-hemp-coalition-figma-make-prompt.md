# Louisiana Hemp Coalition — Figma Make prompt

**How to use:** Paste the block below into Figma Make. Use it to generate or refine the landing (e.g. v3). Adjust section copy or assets as needed; iterate with short, specific follow-up prompts if you need section-level tweaks.

---

## Prompt (copy from here)

```
Create a high-fidelity web mockup for Louisiana Hemp Beverage Coalition (LouisianaHempBeverageCoalition.com), an association.

**Target keywords (weave into headings and body copy where natural):**  
Louisiana hemp beverage association, Louisiana hemp association, Louisiana hemp business

**Style:** Clean, modern, trustworthy (government-adjacent credibility). Light background, restrained palette from the logo (primary + one accent + neutrals), generous whitespace, simple card system, strong typography hierarchy. Make section styles clearly distinct (Policy vs Resources vs Get involved).

**Reference / inspiration:**  
- Competitor 1: [Add URL if available] — Clear top navigation + one primary CTA; scannable "who it's for" strip; concise policy cards  
- Competitor 2: [Add URL if available] — Resources preview with clear links; strong footer with direct policy links and contact info  

**Pages/sections to design:**  
- **Home (one-page landing):** Use a clear section structure: one logical block per section; semantic sections for handoff. Sections in order:  
  1) **Header (sticky):** Logo + links (About, What We Do, Policy, Research, Contact) + primary CTA "Join the Coalition"; highlight active section on scroll  
  2) **Hero:** H1 + 1–2 line subhead + primary CTA "Join the Coalition" (scrolls to form or same flow)  
  3) **Who we represent:** Breadth strip — three items, one line each: Hemp · Alcohol · Grocery (or three small cards). Explicitly show coalition scope near the top  
  4) **Vision / What we do:** 3 bullets or cards max (e.g. Active Engagement, Robust Resources, Powerful Advocacy)  
  5) **Policy priorities:** 3–4 concise cards with icons; each card = title + one sentence + "Learn more"  
  6) **Resources / Updates preview:** 3 items with tags; "View all resources"  
  7) **Newsletter strip:** Full-width teal band — "Stay informed" / "Just get updates" with single email field and CTA "Stay Updated" or "Newsletter only". Label clearly so it is secondary to "Join the Coalition"  
  8) **Get involved / Lead form:** Benefits list (e.g. "Why join the Coalition?" with 3–4 bullets) + lead form. Form: minimal default fields — Full name (one field), Email (required), Phone (optional) with helper text "Optional — only for time-sensitive coordination". Put Company, Role, Industry, Zip, Message behind "Add more details (optional)". Mark required (*) and optional in every label. Encapsulate form in a clear card; add one line under submit: "We'll never share your information." Primary submit: "Submit Your Info" or "Join the Coalition"  
  9) **Footer:** Direct links to Privacy Policy, Terms of Service, Contact. Conventional link text. Logo + short org description  

**Brand assets:**  
- Logo: use the provided Louisiana Hemp Coalition logo (from Notion / project folder)  
- Colors: derive from logo; ensure accessible contrast (teal/green primary, white background, dark gray text)  
- Fonts: not specified — use a highly readable modern pairing (clear body text; confident headings)  

**Goal:**  
Provide information and bring business leaders across Louisiana's hemp, alcohol, and grocery sectors together to shape state and federal policymakers. Simple lead collection and information; one primary action.

**UX requirements (association / local-trust informational):**  
- Navigation: conventional labels; show clear active state for current section (orientation)  
- Homepage breadth: "Who we represent" strip (Hemp / Alcohol / Grocery) near top so visitors see coalition scope immediately  
- Primary CTA: "Join the Coalition" is the single most prominent action (header + hero both go to the same form); newsletter strip is secondary and labeled "Newsletter only" or "Just get updates"  
- Forms: minimal default fields (Full name, Email, Phone optional); mark required (*) and optional in labels; phone helper text "Optional — only for time-sensitive coordination"; Company, Role, Industry, Zip, Message behind "Add more details (optional)"  
- Trust near form: encapsulate form in a clean card; add one line under submit: "We'll never share your information."  
- Footer: direct, standalone links to Privacy Policy, Terms of Service, Contact; conventional link text  
- Mobile: large tap targets (min 44px); form and multi-column sections stack to single column on small viewports; sticky header; link/button hit areas match visible controls  
- Accessibility: semantic heading order (one H1 per page — hero headline; H2 for major sections; H3 for subsections); strong contrast; visible focus states; descriptive button/link labels; meaningful alt text  

**Constraints:**  
Mobile-first; keep copy tight (no long paragraphs); no popups/overlays above the fold; simple 8pt spacing system and consistent component styles; one primary action ("Join the Coalition" = lead form).

Output a complete, production-ready UI suitable for handoff to a developer. Use clear typography hierarchy, consistent spacing, semantic section structure, and one H1 per page with logical heading order (H1 → H2 → H3).
```
